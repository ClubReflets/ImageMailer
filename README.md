# ImageMailer
Script en python permettant d'automatiser l'envoi d'images par courriel

![](doc/images/terminal.png)

## Prérequis 
* [Python 3](https://www.python.org/downloads/)

## Lancement 
Exécuter cette commande dans un invite de commande (terminal)
``` python 
python3 main.py
```
